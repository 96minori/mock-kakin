## MOCK-KAKIN (honkai kakin-emulator)
这是一个基于angular5的崩坏3rd蛋池模拟器。

## Feature
1. 标配补给、装备补给和精准补给  
2. 自定义蛋池配置（任意物品及概率）  
3. 批量抽取配置（可用于验证）  
4. 几乎和游戏一致的界面风格，拥有保底机制并且默认开启。
